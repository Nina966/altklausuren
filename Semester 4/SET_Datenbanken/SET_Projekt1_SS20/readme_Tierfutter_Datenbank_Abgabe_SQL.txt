In der Datei Tierfutter_Datenbank_Abgabe.sql sind die SQL Befehle enthalten um die Datenbank der Gruppe Tierfutter zu erstellen.
Falls sich die Datei nicht als SQL Datei in der Wunschsoftware importieren lässt müssen die Befehle aus dem SQL Dokument kopiert 
werden und als entsprechende Querys in der Wunschsoftware ausgeführt werden.
